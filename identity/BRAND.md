# Codex — Brand

## Name Rationale

**Codex** — a codex is the original bound book of knowledge, and the ancestor of every encyclopedia. The word signals depth, permanence, and interconnection — exactly what WikiMem provides. It also echoes the LLM lineage (Codex was OpenAI's code model) without copying it. One word, scholarly, immediately connotes knowledge.

## Tagline

> Ingest anything. Know everything. Improve forever.

## Colors

Warm black (#141312) background · Indigo (#6366f1) as primary accent (knowledge, depth) · Zinc-300 for body text · Green-400 for freshly improved pages · Amber (#f59e0b) for stale-page warnings. Instrument Serif for wiki headings, Poppins for UI chrome.

## Landing Copy

Codex is the wiki that reads itself. Drop in any file — PDF, markdown, code, transcript — and WikiMem generates interconnected pages, builds a knowledge graph, and keeps everything fresh with an autonomous observer loop. Query it by name, search it by BM25, or call it as an MCP tool from any other agent.
