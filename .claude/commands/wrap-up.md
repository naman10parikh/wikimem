End-of-session protocol:

1. Run all tests (`pnpm test` if available)
2. Run linter (`pnpm lint` if available)
3. Update CONTEXT.md with what was accomplished this session
4. List any unfinished work as TODO items
5. Suggest what to tackle in the next session
6. Write a comprehensive handoff to `.claude/handoff.md` with exact file paths, bug descriptions, and what was deployed vs what wasn't. Update `MAINTAINER-CHECKLIST.md` if any new blocking items for the user were discovered.
7. If there are uncommitted changes, summarize them for a potential commit message
