## What Changed in v0.10.0

Key additions since the v0.9.0/v1.0.0 launch drafts (April 2026):

- **`wikimem observe` command** — run the full observer pipeline from the CLI without starting the web server. Scores every page (24-point scale), surfaces orphans, knowledge gaps, and contradictions, optionally auto-improves weak pages under a configurable budget cap.
- **Unified quality scorer** — `wikimem improve` now shares the same 24-point scorer as the observer. Both commands report identical quality numbers, eliminating the prior discrepancy where `improve` used a 5-dimension wiki-level heuristic and `observe` used a per-page 24-point scorer.
- **MCP OAuth 2.1 client** — WikiMem can now act as an MCP OAuth 2.1 _client_, connecting to any MCP-compliant remote server. Includes full PKCE S256, resource indicators (RFC 8707), metadata discovery, and Dynamic Client Registration.
- **Discord sync module** — `src/core/sync/discord.ts` with full discord.js v14 integration. Guild → channel → message history, attachment URLs, user mention resolution.
- **44 connectors fully functional** — all cards wired (was 38/44 at v0.9.0). Microsoft 365 / LinkedIn remain coming-soon.
- **KARP-003/007/010/012 observer patterns** — four new automated improvement patterns (auto-categorize, wiki-wide summary, citation scoring, semantic similarity edges).
